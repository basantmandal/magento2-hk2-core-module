## Dependency Requirements

- `magento/framework`: `^103.0.0`
- `php`: `^8.1 || ^8.2 || ^8.3 || ^8.4`

## Module Dependencies

- `Magento_Backend`
- `Magento_Config`
- `Magento_Store`

## Browser Compatibility

Not Applicable (This is a backend base configuration module).

## Notes

The module behaves as a base module with zero direct storefront footprint, and is designed to run efficiently on any standard Magento architecture matching the compatibility matrix.
